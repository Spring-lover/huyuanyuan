## 判断B是否是A的子树

```java
public boolean isSubStructure(TreeNode A, TreeNode B) {
    if(A == null || B == null) return false;
    // 子树
    return helper(A, B) || isSubStructure(A.left, B) || isSubStructure(A.right, B);
}

public boolean helper(TreeNode A, TreeNode B){
    // 说明B树匹配完成
    if(B == null) return true;

    // 说明A树到底了
    if(A == null || A.val != B.val) return false;

    return helper(A.left, B.left) && helper(A.right, B.right);
}
```

## 判断是否是对称的二叉树

```java
public boolean isSymmetric(TreeNode root){
    if(root == null) return true;

    return helper(root.left, root.right);
}

public boolean helper(TreeNode L, TreeNode R){
    if(L == null && R == null) return true;

    if(L == null || R == null || L.val != R.val) return false;

    return helper(L.right, R.left) && helper(L.left, R.right);
}
```

## 二叉搜索树的后序遍历序列
后序遍历 为 （左 右 根）又因为是二叉搜索树所以 左子树 < 根 < 右子树
当 i >= j 时代表没有子节点直接返回true

```java
public boolean verifyPostorder(int[] postorder) {
    return helper(postorder, 0, postorder.length - 1);
}

public boolean helper(int[] postorder, int i, int j){
    if(i >= j) return true;
    int p = i;
    while(postorder[p] < postorder[j]) p++;
    int m = p;
    while(postorder[p] > postorder[j]) p++;
    return p == j && helper(postorder, i, m - 1) && helper(postorder, m, j);
}
```

## 将二叉搜索数转化为双向链表

public Node treeToDoublyList(Node root){
    if(root == null) return null;
    dfs(root);

    head.left = pre;
    pre.right = head;

    return head;
}

public void dfs(Node cur){
    if(cur == null) return ;

    dfs(cur.left);
    // pre记录cur左侧的结点 
    if(pre == null) head = pre;
    else pre.right = cur; 

    cur.left = pre;

    pre = cur;

    dfs(cur.right);
}
